Python

A linguagem Python está entre as 3 linguagens mais populares no mundo. Essa popularidade se dá principalmente por sua característica de exigir poucas linhas de código e permitir uma leitura fácil, além de ter várias bibliotecas do mundo de Data Science, Machine Learning, desenvolvimento web, automação e jogos. Na Formação Python, você vai aprender a versão 3 dessa linguagem na prática, aplicando orientação a objetos em seu código e boas práticas de programação. Você também vai entender como tratamos erros e conhecer diversas funcionalidades para tratar os diferentes tipos de coleções. Veremos também como ler e escrever diferentes formatos de arquivos usando Python.

Aprenda Conosco:

  • Conhecer a linguagem
  • Aprofundar na orientação a objetos
  • Trabalhar com formatos de dados
  • Fazer testes automatizados

  • Aulas online ao vivo
